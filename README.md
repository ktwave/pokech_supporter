# PokeCh Supporter

Pokémon Championships 向けの補助デスクトップアプリです。
キャプチャ映像から OCR・画像認識でチーム情報を読み取り、対面サポートの参照や、プロファイル別の対面統計をローカルに蓄積します。

## 主な機能

- **映像**: キャプチャデバイスまたは動画ファイル（ドラッグ＆ドロップ）の再生
- **敵パーティ**: タイマー帯の自動スキャン、手動での再スキャン（`T`）
- **自選出・対面選出**: 選出画面のトリガー検知、ターン開始検知との組み合わせでバースト読取
- **対面サポート**: 左パネルに対面ポケモンのデータ（わざ・持ち物・特性・性格・能力ポイント など）を表示
- **統計プロファイル**: 試合ごとの出現／選出／初手を記録（メニュー「プロファイル」で切替・新規作成）
- **ショートカット**: スクリーンショット `S`、録画 `R`、敵 PT 手動フルスキャン `T`
- **設定メニュー**: キャプチャデバイス・スクショ／録画の保存先・照合閾値などを編集（`QSettings` に保存）。終了時のウィンドウサイズも保存され、次回復元されます。

## 動作環境・依存関係

- Windows を想定（`Config` 内の FFmpeg / Tesseract パスが `.exe` 前提）
- **Python 3** ＋ 次のパッケージが必要です（`requirements.txt` に記載のもの以外も使用しています）  
  - PySide6、OpenCV（`cv2`）、NumPy、BeautifulSoup4、pytesseract など

```bash
pip install -r requirements.txt
pip install PySide6 opencv-python numpy pytesseract
```

同梱想定の **Tesseract**・**FFmpeg**・**テンプレ画像**・**アイコン**は `resources/` 以下を参照します。`backend/config/constants.py` の `BASE_DIR` は `os.getcwd()` のため、**リポジトリのルートをカレントディレクトリにして起動**してください。

## 起動方法

```bash
cd <リポジトリのルート>
python front/main_window.py
```
